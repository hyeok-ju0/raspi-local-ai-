### 2. Llama 3.2 3B 설치
```bash
ollama pull llama3.2:3b
```
- 설치 완료, 정상 작동 확인
- 한국어 입력 시 다국어 혼용 현상 있음 → 영어 사용 권장
- 영어 입력 시 정상 작동 확인

### 3. alias 등록
`~/.bashrc`에 아래 alias 추가:
```bash
alias ai='ollama run llama3.2:3b'
alias ai-smart='ollama run qwen3:8b'
```
- `ai`: 일상/간단한 작업용 (빠름)
- `ai-smart`: 복잡한 추론/한국어 작업용

### 4. SSH 키 인증 설정
- 노트북 PowerShell에서 ed25519 키 생성 (`C:\Users\fyfif\.ssh\id_ed25519`)
- 공개키를 라즈베리파이 `~/.ssh/authorized_keys`에 등록
- 키 인증 정상 작동 확인 (패스프레이즈 설정됨)

### 5. 부팅 시 Open WebUI 자동 실행
```bash
sudo systemctl enable docker
sudo docker update --restart=always open-webui
```
- 재부팅 후 http://192.168.0.216:8080 자동 접속 확인
- Llama 3.2 3B, Qwen3 8B 모두 정상 인식 확인

### 6. Syncthing 파일 동기화 설치 및 설정

**라즈베리파이**
```bash
sudo apt install syncthing -y
sudo systemctl enable syncthing@hyeogju
sudo systemctl start syncthing@hyeogju
```
`~/.local/state/syncthing/config.xml` 수정 내용:
- GUI 대기 주소: `127.0.0.1:8384` → `0.0.0.0:8384` (외부 접근 허용)
- `relaysEnabled`: `true` → `false`
- `globalAnnounceEnabled`: `true` → `false`
- GUI 인증 아이디/비밀번호 설정 완료
- 연결 설정: NAT 통과 비활성화, 내부 탐지만 활성화

**노트북 (Windows)**
- Syncthing v2.1.1 설치
- 설치 시 Relays false로 설정
- GUI 인증 아이디/비밀번호 설정 완료

**기기 연결 및 폴더 동기화**
- 노트북 ↔ 라즈베리파이 기기 연결 완료
- 동기화 폴더 설정:
  - 노트북: `C:\Users\fyfif\Sync`
  - 라즈베리파이: `~/Sync`
- 파일 동기화 테스트 완료 ('새로운 세션에 넣을 내용.txt' 전송 확인)
